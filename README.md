# Organia UI Task

## Project overview
Single-page dashboard UI built with Next.js App Router. It includes a fixed desktop sidebar, a collapsible mobile sidebar, header actions, tab navigation, forms, and an embedded Leaflet map component.

## Technology stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Leaflet / react-leaflet
- lucide-react icons

## Setup instructions
1) Install dependencies
```bash
npm install
```
2) Run the dev server
```bash
npm run dev
```
3) Open http://localhost:3000

## Known limitations
- Data is static (no API integration yet).
- Map tiles require network access.
- No authentication/authorization flows implemented.
